# Contributing

Thank you for helping improve this public-interest AI language protocol.

## How to Suggest Changes

You may contribute by:

1. Opening an Issue with comments or concerns.
2. Starting a Discussion for broader review, if Discussions are enabled.
3. Submitting a Pull Request with exact wording changes.

## Main Branch Protection Principle

The `main` branch represents the current author-approved version.

Please do not expect direct changes to be accepted automatically. All suggested edits require review before merging.

## Evidence and Language Standards

Contributors should:

- Use calm, respectful language.
- Label memory, testimony, records, evidence, and inference separately.
- Avoid personal attacks, unsupported accusations, private sensitive material, or confidential evidence.
- Keep the protocol humble, lawful, international, business-English, and public-safe.

## AI-Assisted Contributions

AI-assisted edits are welcome if they are clearly reviewed by a human contributor.

When practical, note which AI system helped draft or review the suggestion.

Example:

> This edit was suggested after review with Gemini / Claude / Copilot / Grok / Perplexity / another AI assistant.

## Public Reasoning Trace

This repository may include public reasoning summaries, decision logs, and machine-readable rules. It should not include private chain-of-thought, confidential ACIR evidence, raw files, witness identities, or sensitive family/health testimony.

## Author Review
